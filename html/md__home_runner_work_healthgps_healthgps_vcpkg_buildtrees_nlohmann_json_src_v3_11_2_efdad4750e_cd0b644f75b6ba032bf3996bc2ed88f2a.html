This enumeration is used in the `sax_parse` function to choose the input format to parse:

json : JSON (JavaScript Object Notation)

cbor : CBOR (Concise Binary Object Representation)

msgpack : MessagePack

ubjson : UBJSON (Universal Binary JSON)

bson : BSON (Binary JSON)

bjdata : BJData (Binary JData)
